# SilverAutomation
> To be able to run this extension make sure you have already installed the latest python version: https://www.python.org/downloads/
> open a cmd and run this command:         pip install psutil robloxapi pyautogui pillow

Once you are done drag SilverAuto.py to the mewt sniper folder and run SilverAuto.py, wait for it to load all the needed stuff and then it will run mewt automatcally
SilverAutomation gives you FULL control on mewt with some commands that are easy to follow!
